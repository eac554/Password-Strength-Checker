## Password-Strength-Checker

A client-side password strength checker that evaluates the user's demo password security, in real-time, and an estimation of how long it will take to crack

## Features

- Real-time password strength evaluation
- Estimation of time it takes to crack user's demo password
- Utilizes common patterns and weaknesses to estimate the strength of the inputted password
- Client-side and Anonymous (nothing is stored in a server and everything is completely local)
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How it Works

Passwords are evaluated on:
- Length
- Variety of characters
- Common Patterns and dictionary words
- Repitition

## Usage

Visit https://eac554.github.io/Password-Strength-Checker and start testing demo passwords (never a real password) for real-time analysis

## Local Usage
1. Clone the repository
2. Open the "index.html" file in a browser and test passwords without an internet connection locally

## Privacy
All password checks are done completely client-side in the user's browser. Nothing is stored or saved in a server anywhere
